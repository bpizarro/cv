Experiencia académica


LiquiFacil (Segundo semestre 2024)

Proyecto de informática que consistía en realizar un software pensado para una empresa. Se pensó en algo relacionado en contabilidad, por lo que se hizo un software para calcular y emitir liquidaciones de sueldos.

Se verificó todos los pasos que conlleva realizarlo mediante el diagrama BPMN.

Se hizo una conexión a través de la base de datos para solicitar la ficha de cada trabajador y obtener los datos necesarios para los calculos para la liquidación.

El proyecto se llevó a cabo mediante PHP y HTML, con conexión a MySQL para las bases de datos.

Debido a problemas no se pudo terminar el proyecto.




Metro Dijkstra(Segundo semestre 2024)

Proyecto que se realizó a través de la teoría de grafos, el cual se tuvo de referencía el metro de Santiago de Chile para obtener la ruta más corta.

Para realizarlo se tomo las estaciones del metro como vértices y las combinaciones posibles por cada linea para poder obtener el tiempo en el que se demoran.

Se realizó en Python para más facilitar el proyecto, utilizando librerías para visualizar la ruta
